# nmTeam User Center 更新日志

亲爱的用户，您期盼已久的新版本 nmTeam 用户中心来了！新版本 nmTeam 用户中心是新一代的账户系统，为不同搭载 nmServices 设备的智能化、互联与协同提供了统一的接口，为您带来简捷、流畅、连续、安全可靠的 nmServices 体验。
